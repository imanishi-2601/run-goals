# Rungoals

## サイト概要
  市民ランナー向けの目標共有・練習記録コミュニティサイト

### サイトテーマ
  目標設定、練習記録の投稿、同じ目標を持つランナーの練習記録閲覧を通じて、
  ユーザー同士が互いに刺激を受けながら成長できるサイト。
​
### テーマを選んだ理由
私は市民ランナーとしてマラソン大会への参加や日々の練習を行っています。
ランニングは基本的に一人で行う時間が長く、
モチベーション維持が難しいと感じることがありました。
目標に向けたトレーニングでは、
- 他のランナーがどのような練習をしているか
- 同じ目標のランナーの努力量

を知ることで大きな刺激になると考えました。
そのため
- 目標を軸にランナーが繋がる
- 練習記録を振り返られる
- 継続のモチベーションを高められる

サービスを作りたいと考えました。
市民ランナーの継続支援と成長の可視化を目的としたコミュニティサイトにしていきたいです。
​
### ターゲットユーザ
- マラソン、ランニングを趣味としている市民ランナー
- 目標タイムがある人
- 練習のモチベーション維持に課題を感じている人
- 他のランナーの練習内容を参考にしたい人
​
### 主な利用シーン
- 日々の練習後に記録を投稿するとき
- 自信の練習記録振り返るとき
- 同じ目標を持つランナーの練習内容を確認したいとき
- 大会に向けて練習計画を作成するとき
