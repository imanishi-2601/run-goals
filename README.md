# Rungoals

## サイト概要

### サイトテーマ
  市民ランナーが目標と練習記録を共有し、<br>
  他ランナーの記録を閲覧できるランニングコミュニティサイト
​
### テーマを選んだ理由
私は市民ランナーとしてマラソン大会への参加や日々の練習を行っています。<br>
ランニングは基本的に一人で行う時間が長く、
モチベーション維持が難しいと感じることがありました。<br>
目標に向けたトレーニングでは、
- 他のランナーがどのような練習をしているか
- 同じ目標のランナーの努力量

を知ることで大きな刺激になると考えました。
そのため
- 目標を軸にランナーが繋がる
- 練習記録を振り返られる
- 継続のモチベーションを高められる

サービスを作りたいと考えました。<br>
市民ランナーの継続支援と成長の可視化を目的としたコミュニティサイトにしていきたいです。
​
### ターゲットユーザ
- マラソン、ランニングを趣味としている市民ランナー
- 目標タイムがある人
- 練習のモチベーション維持に課題を感じている人
- 他のランナーの練習内容を参考にしたい人
​
### 主な利用シーン
- 日々の練習後に記録を投稿するとき
- 自信の練習記録振り返るとき
- 同じ目標を持つランナーの練習内容を確認したいとき
- 大会に向けて練習計画を作成するとき
​
